DOM Document Object Model

Le DOM est une representation de la page que le navigateur est en train d'afficher

Le DOM est sous la forme arborescence de noeud (Node) dont windows.document est la racine.

Node (nœud) est une classe abstraite de laquelle de nombreux objets de type DOM API héritent, leur permettant d'être traités de manière similaire et souvent interchangeable. Étant une classe abstraire, il n'existe pas d'objet qui soit directement un Node. Tous les objets implantant les fonctionnalités de Node sont des instances d'une de ses sous-classes, dont les plus notables sont Document, Element et DocumentFragment.

Les noeud (node) du DOM peuvent etre de differrent type.

element Classe de base generique de tout les element de document voici la liste de tout les type de noeud

Retourne une DOMString (chaine de caractères) représentant l'URL de base du document contenant le Node. En HTML, il correspond au protocole, au nom de domaine et à la structure du répertoire, jusqu'au dernier /.:document.baseURI

(Non disponible pour le contenu Web). L'objet en lecture seule nsIURI représentant l'URI de base pour l'élément.:document.baseURIObject

Renvoie un NodeList contenant tous les enfants de ce nœud. NodeList signifie que si les enfants du Node changent, l'objet NodeList est automatiquement mis à jour.:document.childNodes

Renvoie un Node représentant le premier nœud enfant direct de ce nœud ou null s'il n'a pas d'enfant:document.firstChild

Retourne un Node représentant le dernier nœud enfant direct de ce nœud ou null s'il n'a pas d'enfant.:document.lastChild

Retourne un Node représentant le nœud suivant dans l'arbre ou null s'il n'y en a pas.:document.nextSibling

Retourne une DOMString (chaîne de caractères) contenant le nom du nœud. La structure du nom diffère du type de nœud. Par exemple, un HTMLElement contiendra le nom de la balise correspondante, comme 'audio' pour un HTMLAudioElement (en-US), un nœud Text doit avoir la chaîne '#text' ou un Document doit avoir la chaîne '#document'..:document.nodeName

Retourne un unsigned short (non signé court) représentant le type du nœud.:document.nodeType
